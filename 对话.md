## 使用GitLab CI/CD的步骤：
### 1. 配置GitLab Runner
在Windows或Linux机器上安装并注册GitLab Runner（参考README中的详细步骤）
docker run -d --name louwen-runner --restart always -v /srv/gitlab-runner/config:/etc/gitlab-runner -v /var/run/docker.sock:/var/run/docker.sock gitlab/gitlab-runner:latest
docker exec -it louwen-runner gitlab-runner register

### 2. 推送代码到GitLab
git add .
git commit -m "Add C++ GUI app with GitLab CI/CD"
git push origin main
### 3. 自动构建
- GitLab CI/CD会自动触发Pipeline
- 在 "CI/CD" -> "Pipelines" 查看构建状态
- 构建完成后下载 CppGuiApp-Windows.zip
### 4. 创建发布版本
git tag v1.0.0

可用的Executor选项：

- shell - 直接在宿主机上执行命令（您选择的）
- docker - 在Docker容器中执行任务
- docker+machine - 使用Docker Machine动态创建容器
- kubernetes - 在Kubernetes集群中执行
- ssh - 通过SSH连接到远程机器执行
- parallels/virtualbox - 在虚拟机中执行
